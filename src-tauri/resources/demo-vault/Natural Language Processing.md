# Natural Language Processing

NLP (Natural Language Processing) is a core subfield of AI focused on enabling computers to understand and generate human language.

## Historical Phases

1. **Rule-based Era** (1950-1990): Hand-crafted grammar rules
2. **Statistical Era** (1990-2013): n-grams, CRF, SVM
3. **Deep Learning Era** (2013-2017): RNN, LSTM, Word2Vec
4. **Large Model Era** (2017-present): [[Transformer Architecture]] dominates everything

## Core Tasks

- Text Classification → [[BERT]] excels
- Text Generation → [[GPT Series]] excels
- Information Retrieval → [[Retrieval-Augmented Generation]]
- Question Answering → RAG + Agent
- Knowledge Extraction → [[Knowledge Graphs]] construction
