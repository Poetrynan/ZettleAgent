# Zettelkasten Method

Niklas Luhmann's Zettelkasten ("slip box") is a **networked** knowledge management system.

## Core Principles

1. **Atomicity**: Each note contains exactly one idea
2. **Autonomy**: Each note is self-contained and understandable on its own
3. **Connectivity**: Cross-references build a web of knowledge between notes
4. **Emergence**: The value of knowledge comes from connections, not individual notes

## Luhmann's Practice

Luhmann accumulated **90,000 notes** over his lifetime, publishing **70 books** and **400+ academic papers**. He credited the system for his productivity — "I never force myself to do anything I don't feel like doing. Whenever I don't know what to do next, I write a note."
