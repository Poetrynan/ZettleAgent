Global Search + Version Timeline + GPU Graph

Ctrl+Shift+F opens a VS Code-style global search panel with file-grouped results and keyword highlighting. Smart routing: hybrid search when embeddings are ready, FTS5 fallback otherwise. Plus: note version snapshots in SQLite with character-level Diff comparison (split/inline view, red-green highlighting). The Knowledge Graph runs on PixiJS v8 (WebGL) with Web Worker physics — D3-force runs entirely off the main thread using Float32Array transferables for zero UI blocking. Nodes drift gently like boats on water. Includes weighted PageRank, Louvain community detection, BFS local graph, orphan filter, and neighbor spotlight highlighting — all at 60fps with 2000+ nodes.

ctrl+shift+f · hybrid/fts · diff · snapshots · pixijs v8 · 60fps · pagerank
